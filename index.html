Linux very basic shortcuts

Basic commands:

  • alt + f1 = opens the system menu
  • alt + f2 = opens the command launcher
  • alt + f4 = closes the focus window
  • shift + del = permanently deletes a file without sending it to the Trash
  • alt + tab = cycles between open applications
  • ctrl + z = undo
  • ctrl + shift + z = redo
  • ctrl + alt + l = locks the screen
  • ctrl + backspace = deletes an entire word in many text editors
  • ctrl + a = selects all
  • ctrl + c = copies the selection
  • ctrl + x = cuts
  • ctrl + v = pastes
  • shift + arrow keys = keys selects
  • ctrl + right - left arrow keys = moves the cursor a word at a time
  • ctrl + up-down arrow keys = either scrolls up and down a line at a time without moving the cursor
